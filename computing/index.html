Packages

  • LinterpOpenCL

    This package provide a very fast linear interpolation LinterpOpenCL is a header-file-only package doing linear interpolation designed for scientific applications programmed with OpenCL to utilize GPUs. Any C++ program can easily call this package. Both CPU (without OpenCL) and GPU versions are available.

  • Replication of Adaptive Sparse Grid of Brumm and Scheidegger (2017, Econometrica)

    This is another C++ implementation of Brumm and Scheidegger (2017, Econometrica). The code of the original paper had the interpolation part, but the how to adaptively construct the grids was missing in the original code. In this replication, the researcher just need to provide his function to be approximated, and all the things are provided. Only CPU version is available, and OpenCL GPU version is also available on request.

  • C++ version of Judd, Maliar, and Maliar (2011, QE)

    This is C++ implementation of Judd, Maliar, and Maliar (2011, Quantitative Economics), which proporses an efficient algorithm of integrating a function with normally distributed (and possibily correlated) shocks. I frequently use this when solving consumption and savings problem to construct expected value functions.
